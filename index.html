Let Const Exercises

What is the difference between var and let? var is function scoped, let is block scoped. var variable declarations are run first while let variable declarations are not.

What is the difference between var and const? var can be reassigned, const cannot. var is function scoped, const is block scoped. var variable declarations are run first while const variable declarations are not.

What is hoisting? Hoisting is when the variable is declared before the variable is initialized.
